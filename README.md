# Web-programozás 1 beadandó

## Téma
Magyarországi foglalkoztatási és lakossági statisztikai webalkalmazás

## Funkciók
- Főoldal bemutatkozó felület
- JavaScript CRUD
- React CRUD
- SPA két React miniappal
- Fetch API CRUD
- Axios CRUD
- OOJS grafikus alkalmazás

## Felhasznált technológiák
- HTML5
- CSS3
- JavaScript
- React
- Fetch API
- Axios
- PHP
- MySQL
