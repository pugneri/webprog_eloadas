Projektbemutató

Ez a webalkalmazás magyarországi régiók, megyék, lakossági adatok és foglalkoztatási statisztikák kezelésére készült. A projekt célja, hogy többféle webfejlesztési technológiát egy közös témán keresztül mutasson be.

Az alkalmazás külön oldalakra bontva szemlélteti a JavaScript CRUD, React CRUD, SPA, Fetch API, Axios és objektumorientált JavaScript megoldásokat.

JavaScript CRUD megnyitása Fetch API megnyitása

Projekt célja

Egy egységes témára épülő beadandó elkészítése, amely a tárgyban tanult kliens- és szerveroldali technológiákat is bemutatja.

Adatfókusz

Régiók, megyék, lakosságszám és foglalkoztatási adatok elemzése, megjelenítése és kezelése.

Főbb statisztikai adatok

A projekt alapját valós, táblázatos formában megadott magyarországi regionális, megyei, lakossági és foglalkoztatási adatok adják.

Régiók száma

7

A forrásadatok 7 magyarországi régiót tartalmaznak.

Megyék száma

20

A projekt 20 megye adataival dolgozik Budapesttel együtt.

Összes lakosság

10 090 330

A betöltött lakossági adatok összege.

Foglalkoztatási rekordok

20

Minden megyéhez külön foglalkoztatási rekord tartozik.

Az alkalmazás menüpontjai

JavaScript CRUD

Tömbben tárolt megyei adatok kezelése.

React CRUD

Régiók kezelése React állapotkezeléssel.

SPA

Két kisebb React alkalmazás egyoldalas megoldásban.

Fetch API

Adatbázisban tárolt foglalkoztatási rekordok kezelése JavaScript és Fetch API segítségével.

Axios

React alapú CRUD felület szerveroldali adatbázissal és Axios kommunikációval.

OOJS

Objektumorientált JavaScript grafikus megoldás osztályokkal és öröklődéssel.

Miért ezt a témát választottuk?

A régiókhoz, megyékhez, lakossághoz és foglalkoztatáshoz kapcsolódó adatok jól strukturáltak, könnyen relációs adatbázisba szervezhetők, és alkalmasak arra, hogy a beadandó minden részfeladatát ugyanarra a témára építsük fel.

Ez azért előnyös, mert az egész projekt egységes marad, a dokumentáció átláthatóbb lesz, és minden menüpont ugyanazt az adatvilágot dolgozza fel más technológiai megközelítéssel.
